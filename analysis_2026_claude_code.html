Debates entre creadores

Code vs Cowork: destinos opuestos

"Cowork reemplazara a Code"

Brock Mesarich argumenta que Cowork evoluciona rapido y podria reemplazar Code en 3-12 meses para la mayoria de tareas, gracias a su interface superior y sandbox seguro.

"Code es irreemplazable"

Multiples videos enfatizan que Code ofrece terminal access, APIs, agent teams y control total de la maquina que un sandbox como Cowork no puede igualar.

Veredicto: Usa ambos. Code para engineering, Cowork para admin. No son competidores.

OpenClaw: heroe o villano

"El asistente AI del momento"

Adopcion masiva, community marketplace con miles de extensiones. El standard de facto para asistentes personales AI.

"Security nightmare"

Cole Medin lo describe como vulnerable: remote code execution, packages maliciosos en marketplace. Recomienda replicar su logica en Claude Code.

Veredicto: NO uses OpenClaw directamente. Replica su logica (memory, heartbeat) en tu setup con Claude Code.

Mencionados en 3+ videos

Warnings comunes

"Blast radius" de tareas grandes

"Instead of asking for a massive redesign at once, give focused, medium-sized tasks and validate at each step." Regla: nunca toques mas de 3-5 archivos a la vez.

Nunca pegues secrets en chat

"Never paste secret keys, API tokens, or SSH credentials directly into a chat." Usa siempre .env + .gitignore.

Context decay (~mensaje 30)

"Around message 30 it just starts ignoring things. It rewrites code it already wrote. It feels like it forgot everything. Well it did." Sesiones cortas + commit + nueva sesion.

Permisos que frenan el flow

"Agents stopping constantly to ask for permissions." Pre-aprueba tools comunes en settings.json o usa Auto Mode (nuevo, 24 marzo).
